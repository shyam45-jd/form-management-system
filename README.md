## 🚀 Form Management System

A dynamic and scalable application that allows users to create, manage, and reuse customizable form templates with real-time validation.

---

## ✨ Features

* Create dynamic forms
* Reusable form templates
* Multiple input types (text, checkbox, dropdown)
* Form validation
* Store and manage responses

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Spring Boot
* Database: MySQL

---

frontend/

```

## ⚙️ Setup & Run

### Frontend

```
